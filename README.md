# How To Run This Application

- Split terminal
- In the first terminal:
  - <code>cd server</code>
  - <code>npm i</code>
  - <code>nodemon server.js</code>
- In the second terminal:
  - <code>cd client</code>
  - <code>npm i</code>
  - <code>npm run dev</code>
- Run localhost on port 5173 on different browsers
- Enjoy the Game!
